# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
- ローカルリポジトリ：自分のPCにあるGitの履歴、ブランチ一式。
- リモートリポジトリ：GitHubなどサーバー上にある共有用のリポジトリ。


## プッシュとマージの違いは何でしょうか？
- プッシュ：ローカルのコミットをリモートに送ること
- マージ：ブランチの変更を別のブランチに合流させること


## コミットとプッシュの違い
- コミット：変更をローカルリポジトリに保存すること
- プッシュ：ローカルリポジトリのコミットをリモートリポジトリに送信して共有すること


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
後から見返して、何をしたかが一瞬でわかるのが最適です。


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- ローカルでマージするフローは自分のPCでmergeして内容を合流させるため、時短で手軽だがレビューがない
- プルリクエストでマージするフローはリモート上で差分を見せてレビューをしてから合流させるため、時間がかかるが、安全


## コンフリクトを起こしてしまった場合、どう対処すべきですか？
git statusでどのファイルが衝突しているかを確認し、衝突箇所を正しい形に修正する。
